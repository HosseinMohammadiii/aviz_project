## 📖 Description

### What is Aviz?

Aviz is a Flutter-based mobile application developed for the real estate market. It provides a simple and intuitive platform for buying, selling, and renting properties, allowing users to create, manage, and search property listings with ease.

### ✨ Features

- 🔐 User authentication (Sign Up & Log In)
- 🔔 Push and local notifications (Firebase & Local Notifications)
- 🏠 Create property listings for sale and rent
- 📍 Browse property listings by province and city
- 🔍 Search and filter property listings
- 📄 View detailed property information
- 📞 Contact property owners via phone call or SMS
- 📝 Publish property listings with detailed information and images
- 🖼️ Upload images for property listings
- 🗑️ Delete your published property listings
- 👤 View and manage your published listings
- 🕒 View recently viewed property listings
- ✏️ Edit your profile information
- 📱 Responsive design for all screen sizes

### ⚡ State Management

- **BLoC**
